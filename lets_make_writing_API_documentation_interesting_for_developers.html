MSON - Markdown Syntax for Object Notation

Plain-text, human and machine readable, description format for describing data structures in common markup formats such as JSON, XML or YAML.

Facilitate the discussion & validation of data structures

Developed by Apiary as a part of the API Blueprint syntax

Typical API Docs = Route + Json Api Blueprint = Markdown + MSON
